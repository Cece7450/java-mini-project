# Java Programming Exercises

This repository contains various Java applications focusing on logic, string manipulation, and object-oriented concepts.

### Projects Included:
1. Student Management System
   - A system to manage student data using Java, focusing on data structure and CRUD logic.
2. Perimeter Evaluation
   - Calculations for geometric shapes and coordinate geometry.
3. DNA Gene Evaluation
   - Advanced string manipulation to find and evaluate gene sequences within DNA strings.

### Skills Demonstrated:
* Object-Oriented Programming (OOP)
* String Manipulation & Algorithms
* Problem Solving & Logic Development
